# 更新内容一览
## 更新了 UI ToolKit，支持了 World Space UI
- 这是更新的 UI 框架，老框架就是在线教程中常见的 UGUI
## Graph Toolkit
- 提供了 API 来制作图表工具
- 或许新项目的 Flow 可以用上这个工具
## Cloud Diagnostics
- 分析玩家，得到玩家运行的Error
- Free
- 用户分析
## [Unity AI Roadmap](https://unity.com/cn/roadmap#ai-general)
- Unity Points
- Generators 
	- Sprites、Sounds、Animations
